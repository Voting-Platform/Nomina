# Project Dependencies

This project relies on modern, production-ready libraries for both the backend and frontend.

## 🚀 Backend (Server) Key Dependencies

The server-side framework is optimized for scalability and performance.

### Core Framework & API
- **express**: Fast, minimal web framework for Node.js.
- **mongoose**: MongoDB object modeling tool.
- **joi / zod**: Schema-based validation for request bodies and environment variables.
- **passport / passport-jwt**: Robust authentication middleware with JWT support.
- **cors / helmet**: Essential security and resource-sharing middlewares.

### Utilities & Processing
- **aws4**: AWS request signing for cloud operations.
- **bwip-js**: Barcode generator for Node.js and browsers.
- **exceljs / json2csv**: Handling Excel and CSV data exports/imports.
- **handlebars**: Semantic templates for email and reporting.
- **moment / date-fns**: Sophisticated date manipulation.
- **node-schedule**: Task scheduling for repetitive backend jobs.
- **nodemailer**: Email delivery for transactional systems.
- **puppeteer / jsreport**: Advanced PDF and report generation.
- **sharp**: High-performance image processing.
- **tesseract.js**: OCR library for extracting text from images.
- **winston / morgan**: Enterprise-level logging system.

---

## 🎨 Frontend (Client) Key Dependencies

The client-side uses a cutting-edge technological stack.

### UI & Styling
- **React (v19)**: The foundational UI library.
- **TailwindCSS (v4)**: Modern, utility-first CSS framework.
- **Framer Motion**: State-of-the-art animation library for fluid transitions.
- **Radix UI**: Primitive UI components for accessible, high-quality design systems.
- **Lucide React**: Clean, consistent icon set.

### State Management & Data Fetching
- **Zustand**: Minimalist and powerful global state management.
- **TanStack Query (React Query)**: Data-fetching, caching, and state synchronization.

### Data Manipulation & Tables
- **TanStack Table (React Table)**: Headless UI for building powerful, scalable tables.
- **React Hook Form**: Performant, flexible, and extensible forms.
- **Zod**: TypeScript-first schema validation for forms and API responses.

### Specialized UI Components
- **Dnd-kit / Pragmatic Drag and Drop**: Sophisticated drag-and-drop interactions.
- **Recharts**: Data visualization using modular charting components.
- **Leaflet / React-leaflet**: Performant mapping and spatial visualizations.
- **React Day Picker**: Date and range selection components.
- **React MD / Remark / Rehype**: Robust markdown rendering with KaTeX support.
